# gBuster
Simple Non-recursive directory bruteforcer

usage:
------
```
 python3 gBuster.py -u [url] -w [wordlist]
```
example:
--------
```
 python3 gBuster.py -u https://www.google.com/ -w /usr/share/wordlists/dirb/common.txt
```
